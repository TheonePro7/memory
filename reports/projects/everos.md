## 定位

"Build, evaluate, and integrate long-term memory for self-evolving agents."——让 Agent 记忆"自我组织"并随时间进化。

## 核心能力

| 组件 | 得分 | 备注 |
|---|---|---|
| **EverCore** | **93.05% LoCoMo** | 受生物印记启发的自组织记忆 OS |
| **HyperMem** | **92.73% LoCoMo** | 超图层次记忆，ACL 2026 Main 接收 |
| **EverMemBench** | — | 三层基准（事实回忆/应用推理/个性化泛化） |
| **EvoAgentBench** | — | 纵向成长曲线评测（非静态快照） |

- **Rest API**： localhost:1995 记忆搜索服务
- **Claude Code 插件**：可用作 Claude Code 的记忆后端
- **记忆图谱可视化**：记忆关系浏览器
