## 定位

"AI memory OS for LLM and Agent systems"——不只是记忆存储，而是提供"技能记忆"让 Agent 能跨任务复用技能。

## 核心能力

- **图记忆**：结构化图存储，支持关系遍历
- **多模态记忆**：文本、图片、工具调用轨迹、角色入设
- **记忆立方体**：可组合的知识库——隔离和共享
- **异步调度 (MemScheduler)**：生产级并发
- **混合检索**：FTS5 全文搜索 + 向量搜索
- **智能去重 + 技能演化**：跨任务复用
- **多 LLM 支持**：OpenAI、Azure、Qwen、DeepSeek、MiniMax、Ollama、vLLM
