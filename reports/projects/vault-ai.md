## 定位

用 OpenAI API + Pinecone 向量数据库，让 ChatGPT 拥有文件级长期记忆。用户可以上传文档（PDF/ePub/docx/txt），然后向文档内容提问。

## 核心能力

- 上传多种文档格式构建知识库
- 回答自带文献引用（显示哪个文件哪段话）
- Pinecone 嵌入 + OpenAI embedding
- 仅限 OpenAI + Pinecone 组合
