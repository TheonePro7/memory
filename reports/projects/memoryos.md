## 定位

受操作系统内存管理启发的 Agent 长期记忆。四模块架构：Storage → Updating → Retrieval → Generation，管理短期/中期/长期角色记忆。

## 核心能力

- **LoCoMo 改进**：F1 +49.11%, BLEU-1 +46.18%
- **MCP Server**：`add_memory` / `retrieve_memory` / `get_user_profile`
- **三种记忆窗口**：短期/中期/长期角色记忆
- **全 LLM 支持**：OpenAI / DeepSeek / Qwen / Anthropic / vLLM
- **多种部署**：PyPI / Docker / ChromaDB / Playground
