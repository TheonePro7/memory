## 定位

"User Profile-Based Long-Term Memory for AI Chatbot Applications."——存的不只是对话历史，而是结构化的用户画像。

## 核心能力

- **画像记忆**：结构化用户画像（基本信息/兴趣/人口/心理特征等）
- **时间线事件**：在画像之外维护用户事件时间线
- **可控模式**：开发者自定义画像字段
- **批处理**：离线批处理，在线延迟 < 100ms
- **上下文 API**：将相关记忆打包到 prompt 字符串，可配 token 上限
- **技术栈**：FastAPI + Postgres + Redis + Docker
